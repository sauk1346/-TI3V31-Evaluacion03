## Instalación y Ejecución Local

1. Clonar repositorio

```bash
git clone https://github.com/sauk1346/-TI3V31-Evaluacion03.git
cd -TI3V31-Evaluacion03
```

2. Instalar dependencias

```bash
npm install
```

3. Ejecutar en modo desarrollo

```bash
npm run dev
```
